# App Improvements Plan

## Overview
This document outlines comprehensive improvements to enhance functionality, visuals, and user experience based on requirements.

## Key Requirements Recap

1. **Baseline Recording**: 
   - Caregiver selects condition (Anxiety, Stress, Discomfort)
   - Initialize as zero → Record actual values
   - Store in database as individual personal baselines

2. **Real-time Processing**:
   - Continuous sensor readings from database
   - Compare against baselines
   - Detect exact emotional state
   - Show on app + send notifications

3. **Visuals & Functionality**:
   - Proper sensor readings display
   - Undefeatable backend schema
   - Top-notch backend-frontend connection
